# tex-thesis-skeleton
A more simplified TEX thesis template that compiles without problems.

## Get it running

- Cloning the repository
- then build it 

```
pdflatex main.tex
pdflatex main.tex
```

## Get it running on windows
- Install MiKTeX    https://miktex.org/
- Install Perl      http://strawberryperl.com/ 
- Then build it

```
latexmk -pdf
```
